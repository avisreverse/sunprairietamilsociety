The Thiruvalluvar Library

Classical authority meets editorial wisdom

Design F

The Bento Village

A living community map as dashboard

The entire homepage IS a bento grid — no traditional hero section. Each program is a distinct colored tile with its own identity. Sticky sidebar navigation (collapses to bottom tabs on mobile). Feels like a community town map — you see everything at once.

Asymmetric bento grid Sidebar + bottom tabs Per-program colors Tile "breathing" hovers

Design G

The Sangam Scroll

Landscape poetry as navigational journey

The ancient Tamil Sangam tinai system — 5 emotional landscapes — as full-page scroll sections. Each section has a completely different color palette. Thiruvalluvar couplets appear as full-bleed typographic moments between sections. Dot navigation on the right. Uniquely, irreducibly Tamil.

Bold creative studio — Tamil culture made visible

Design I

The Living Temple

Gopuram as navigational architecture

A CSS/HTML gopuram tower built from program content. Left pillar navigation (vertical nav labels, temple column aesthetic). Mandapa colonnade layout for programs (5 pillars side by side). Stone palette with saffron + kumkum festival accents. Inscription-style achievement display. The most architecturally Tamil of all six.
